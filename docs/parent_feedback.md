# Parent Feedback Template

Use this template to collect first real user feedback after the Firebase Hosting deployment.

Live app: https://test-sg-moe.web.app

## Test Flow

Ask each parent to test on the device they normally use for school research, preferably mobile first.

1. Open the live app.
2. Start from the home page.
3. Go to the recommendation flow.
4. Select filters and view recommendation results.
5. Open the predict panel for at least one school.
6. Note anything confusing, missing, slow, or broken.

## Questions To Ask

- What confused you?
- What information did you expect to see but could not find?
- Did the recommendation results feel understandable and useful?
- Did it work properly on your phone?
- Was any wording unclear or too technical?
- What would you want improved before using this for an actual school shortlist?
